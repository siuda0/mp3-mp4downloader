README (日本語)

🔧 yt-dlp ダウンローダースクリプト

このスクリプトは、yt-dlp ライブラリを使用して、YouTubeからオーディオ（MP3）またはビデオ（MP4）ファイルをダウンロードできます。
yt-dlp のアップデートを自動で確認し、必要に応じてインストールや更新を行い、ダウンロードプロセスをガイドします。

☞ 必要条件

Python 3.6 以降がシステムにインストールされていること。

yt-dlp と動画をダウンロードするためのインターネット接続。

オーディオ抽出（MP3 ダウンロードの場合）のためにインストールされた FFmpeg。

📖 使用指南
1 ステップ2: yt-dlp のインストールまたは更新

スクリプトは yt-dlp がインストールされているか確認します。インストールされていない場合、インストールを求められます。

インストール後、yt-dlp のアップデートを自動的に確認し、必要に応じて更新を実行します。

2 ステップ3: ダウンロード先の選択

ダウンロードファイルを保存するフォルダのパスを入力します（例: C:/Users/user/Documents/BGM）。

スクリプトがそのパスが有効かどうか確認します。

3 ステップ4: 形式を選択

以下の中から選べます：

mp3: オーディオのみをダウンロード。

mp4: ビデオとオーディオを両方ダウンロード。

4 ステップ5: YouTube URL を入力

ダウンロードしたい YouTube 動画の URL を貼り付けます。

スクリプトがダウンロードを開始し、選択したディレクトリにファイルを保存します。

5 ステップ6: 再実行または終了

ダウンロード完了後、スクリプトは以下を要求します：

Enter キーを押して終了。

download を入力して別のファイルをダウンロード。

🔔 注意事項

このスクリプトを使用する際は、YouTube の利用規約を遵守してください。

FFmpeg がインストールされていない場合、MP3 の変換に失敗する可能性があります。FFmpeg をインストールし、システムのパス (PATH) に追加してください。

このスクリプトは責任を持って使用し、ダウンロードする権利があるコンテンツのみに使用してください。
